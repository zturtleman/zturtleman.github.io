A video game engine based on ioquake3.

What is Spearmint?

Spearmint is a heavily modified version of the Quake III Arena engine. Spearmint is directly based on ioquake3.

How Spearmint is different from the original Quake 3 / ioquake3 engine;
  • It is possible to do a hell of a lot more without modifying the engine.
  • New features for the modding APIs and shader files.
  • Split engine and game code into separate projects.
  • More Quake 3 game code improvements.
  • 4-Player Splitscreen support.
  • Not compatible with Quake 3 mods or replay demos.
For people familiar with modding Quake 3 based games;
  • It's possible to extend the entityState_t and playerState_t data structures without changing the engine.
  • Merged UI VM into CGame VM so the CGame<->UI communication is more flexable.
  • Moved a lot of code from server and client to Game and CGame VMs;
    • including the console, chat input overlay, usercmd_t creation, and a lot of bot AI code.
Spearmint has support for some data formats used by other Quake 3 based games:
  • Support for map and model formats from RTCW and ET (missing player model torso frame/axis control).
  • Support for FAKK / Alice FTX images.
  • Partial support for FAKK / Alice BSP map format (missing dynamic entity lights).
  • Partial support for SoF2 / JK2 / JA BSP map format (missing light styles support).

Source Code

The source code for Spearmint and the compatible Quake 3 game code is available under the GPLv3+ (with additional terms) on Github.

  • Spearmint engine: zturtleman/spearmint
  • Quake 3 game code: zturtleman/mint-arena
